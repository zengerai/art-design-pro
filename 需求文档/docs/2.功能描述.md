# 登录及用户权限体系

## 一、登录入口

登录界面使用现有的登录界面：http://localhost:3006/#/auth/login

## 二、登录跳转

登录成功后，不同的用户跳转不同的系统界面

1. 系统后台管理员登录后，跳转到http://localhost:3006/system/dashboard/console
2. 系统管理员和普通用户登录后，跳转到http://localhost:3006/user/dashboard/console

两个页面都直接复制http://localhost:3006/#/dashboard/console页面框架，左侧菜单栏内容以对应的界面都删除

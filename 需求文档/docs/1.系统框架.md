# 项目背景

虚拟币数据分析系统

# 系统框架

## 一、登录管理

### （一）前端功能

直接套用当前项目的登录入口界面：http://localhost:3006/#/auth/login?

## 二、用户体系

分为系统后台管理员（超级管理员）、系统管理员、系统用户

登录成功后，不同的用户跳转不同的系统界面。

- 系统后台管理员登录后，跳转到http://localhost:3006/system/dashboard/console
- 系统管理员和普通用户登录后，跳转到http://localhost:3006/user/dashboard/console

两个页面都直接复制http://localhost:3006/#/dashboard/console页面，保留页面框架，左侧菜单栏内容以对应的界面都删除
